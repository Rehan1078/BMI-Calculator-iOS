# BMI Calculator iOS App

A simple and interactive BMI (Body Mass Index) Calculator built using Swift and UIKit for iOS.

## Features

- Adjust height using slider
- Adjust weight using slider
- Real-time UI updates
- Calculate BMI instantly
- Result screen with calculated BMI
- Simple and clean UIKit interface

## Technologies Used

- Swift
- UIKit
- Storyboard
- Xcode

## BMI Formula

```text
BMI = Weight (kg) / Height² (m²)
```

## Screens

### Home Screen
- Select Height
- Select Weight
- Tap Calculate Button

### Result Screen
- Displays calculated BMI value
- Recalculate option

## Project Structure

```text
BMI Calculator/
│
```

## How to Run

1. Open the project in Xcode
2. Select iPhone Simulator
3. Press `Cmd + R`
4. Run the application
